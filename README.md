# aprenderhaciendo - C#

2014 ecommerce platform created for a specific client in mind, but adapted for re-usability with placeholder images and text. Allows to be customized without much effort for other purposes. It features Bootstrap, HTML, CSS3 and a C#-MSSQL Backend, with an Admin session login for content to be submitted by the user and also an unfinished PayPal and Online Shopping session. Compiled and edited with MS Visual Studio 2013.
